# Products

Productized offerings in the Como Technologies portfolio. Products are the durable artifacts and patterns clients adopt — the output of the Prescribe stage.

- **palette-playbook** — A concrete example of the Como playbook pattern, originally built for InterSystems' cloud and Kubernetes platform team. ADRs capture decisions; guides capture procedures; together they form a living, opinionated reference that evolves with the team. Delivered as a self-hosted static site (mdBook + Docker + nginx) so it lives where teams already operate.

The playbook-as-artifact pattern itself is the product. Each engagement produces a tailored variant — opinionated where the client needs a jumpstart, flexible where they have their own conventions.
