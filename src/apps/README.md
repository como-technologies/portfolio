# Apps

User-facing applications in the Como Technologies portfolio. Apps are where humans interact with the portfolio — authoring assessments, responding to polls, reading capacity reports.

- **assessments** — AI-assisted authoring environment for structured maturity assessments. SMEs co-create assessments with Claude; output is exportable markdown that feeds the Prescribe stage. The starting point of the Como loop.
- **pulse** — Verified-anonymous sentiment polling using cryptographic blind signatures. Captures honest feedback on organizational health during and after adoption. Mathematically guaranteed privacy — responses cannot be linked to identities, even by operators.
- **tuesday** — Team capacity analysis from GitHub activity. Developers self-report relative effort on PRs; tuesday turns that into monthly breakdowns showing where engineering capacity actually goes. Quantitative counterpart to pulse.
